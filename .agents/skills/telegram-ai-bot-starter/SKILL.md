# Навык: Создание Telegram AI-Бота и Оформление GitHub Релизов

Этот навык содержит готовые инструкции и шаблоны для мгновенного развертывания Telegram-ботов и их **официальной публикации на GitHub в виде профессиональных Релизов (в стиле n8n-as-code)**.

## 1. Зависимости
```bash
python -m pip install aiogram google-genai python-dotenv aiohttp-socks
```

## 2. Стандарт публикации на GitHub (Официальные Releases)
При выгрузке любой новой версии на GitHub каждый релиз обязателен к оформлению по следующему стандарту:
1. Создание тега версии (`v1.0`, `v1.1`, `v1.2`...).
2. Публикация официального **GitHub Release** через API со структурой:
   - **Title:** `Название Проекта vX.Y`
   - **Section `## What's Changed`:** Подробный список новых фич и исправлений с маркерными точками.
   - **Section `Full Changelog`:** Прямая ссылка на коммиты этой версии.

## 3. Авто-исправление клавиатурной раскладки (QWERTY ➡️ ЙЦУКЕН)
